This is a project done for CIS330, Professor Hank Childs, Spring 2016 at the University of Oregon.
It takes an input, an image in the .pnm format, and performs a series of manipulations on it. 
The included image in.pnm is what was used for testing. 
After making, to use:
./proj3G some_kind_of_input.pnm the_name_you_want_for_output.pnm
